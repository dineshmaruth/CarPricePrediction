
## overview

This project focuses on predicting car prices using machine learning algorithms
## Dependencies

Python 3
Jupyter notebook

## Data set

quikr_car.csv
## How to Use

pip install
1. numpy
2. pandas
3. scikit-learn
4. matplotlib

Run the Jupyter notebook or VS code.
